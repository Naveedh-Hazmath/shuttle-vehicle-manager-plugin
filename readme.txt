A comprehensive shuttle vehicle management system with owner profiles, vehicle registration, and availability calendar.

== Description ==

Shuttle Vehicle Manager is a complete solution for managing shuttle vehicles, owners, and availability.

**Features:**

* **User Registration/Login via Mobile Number**
  * No email required
  * Mobile number as username
  * Secure password authentication

* **Vehicle Owner Profile Management**
  * Full Name
  * NIC Number
  * Address
  * Driving License Number and Expiry Date
  * Mobile and WhatsApp Numbers
  * Profile status (Pending/Verified)

* **Vehicle Management**
  * Register multiple vehicles with details
  * Upload vehicle documents (RC, Insurance, Emission, Revenue License, Fitness)
  * Upload vehicle images (Interior & Exterior)
  * Mark vehicle availability on calendar
  * View, edit and delete vehicles

* **Admin Panel**
  * View and manage all vehicle owners
  * Verify owners and vehicles
  * Filter vehicles by availability dates
  * Review vehicle documents

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/shuttle-vehicle-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode [svm_dashboard] on any page to display the frontend dashboard.

== Frequently Asked Questions ==

= How do vehicle owners register? =

Vehicle owners can register using just their mobile number and password. No email is required.

= How do I verify vehicle owners? =

As an admin, you can verify vehicle owners from the Vehicle Owners menu in the admin panel.

= Can I see which vehicles are available on specific dates? =

Yes, use the Available Vehicles section in the admin panel to filter vehicles by date range.
